TECNOLOGIAS WEB

La materia se encarga del entorno web, es decir de aplicaciones, foros, portales, sistemas o paginas web. El contenio de la materia es entender los conceptos de los tipos de paginas que se pueden desarrollar, aprender sobre herramientas como github, git entre otras. Aprender sobre CMS y como ayudan a la empresa o proyecto en el que se trabaja a administrar su informacion. Tambien se tocan tecnologias de desarrollo como son los frameworks. Un framework es un marco o esquema de trabajo generalmente utilizado por programadores para realizar el desarrollo de software. Utilizar un framework permite agilizar los procesos de desarrollo ya que evita tener que escribir código de forma repetitiva, asegura unas buenas prácticas y la consistencia del código. Un framework es por tanto un conjunto de herramientas y módulos que pueden ser reutilizados para varios proyectos. Uno de los frameworks más conocidos y utilizados es el .NET Framework de Microsoft para webs.

TOP 10 FRAMEWORKS

# FRAMEWORK LENGUAJE VENTAJAS COLUMNAS
1 REACT JavaScript Altamente escalable y capaz de satisfacer una demanda masiva, como lo demuestra su uso en sitios como Netflix, Facebook, Airbnb y Reddit. Los usuarios de React también pueden usar ReactNative, un marco utilizado para crear aplicaciones en iOS y Android. la arquitectura de React está construida en JavaScript, un aspecto que puede resultar difícil para algunos desarrolladores sin experiencia.
2 ANGULAR JavaScript/TypeScript la popularidad y el uso generalizado de Angular significa que hay una gran cantidad de soporte para desarrolladores disponible. El sitio de Angular está repleto de soporte de documentación las versiones anteriores de Angular no son compatibles con las versiones más nuevas que utilizan TypeScript. JavaScript puede ser difícil de manipular bien para los desarrolladores más nuevos.
3 Django Python El uso del lenguaje Python por parte de Django permite una codificación limpia y eficiente, respaldada por una seguridad superior a la media Realmente no hay muchos. El marco en sí es bastante rígido en algunos frentes, y también puede sentirse un poco hinchado si lo está usando para proyectos de desarrollo más pequeños.
4 Ruby on Rails Ruby ideal para principiantes, especialmente para aquellos que pueden no ser eficientes con los marcos de JavaScript. Viene con todo lo que uno necesita para desarrollar una aplicación web basada en bases de datos. Las aplicaciones Ruby requieren mucho esfuerzo para implementarlas y usarlas en un entorno de producción. También hay una gran curva de aprendizaje cuando se profundiza en el marco.
5 LARAVEL PHP es fácil comenzar y su biblioteca de tutoriales de Laracasts es un recurso enorme, lleno de videos que cubren todo, desde PHP hasta otras tecnologías frontend compatibles con Laravel. Laravel también viene con soporte API. No es el mejor marco para proyectos más grandes. Los usuarios avanzados pueden encontrarlo un poco limitante.
6 ASP.net C# productiva, rápida y sorprendentemente poderosa. como su nombre lo indica, es solo para aplicaciones .NET. El aspecto solo para PC también es limitante.
7 EXPRESS JavaScript Express es uno de los marcos más flexibles y mínimos disponibles, y su sistema de complementos, la compatibilidad con otros marcos y la implementación de la pila MEAN indican que claramente será un marco clave en los próximos años. La arquitectura suelta y no guiada de Express puede ser abrumadora para los principiantes, ya que no hay una forma definida de usarla.
8 VUE JavaScript con solo 20 KB, este es uno de los marcos más ligeros que existen. Es fácil de usar para principiantes, lo suficientemente versátil para usuarios avanzados y se puede utilizar para crear plantillas y componentes. Google y Facebook actualmente no son compatibles con Vue, pero se espera que esto cambie en el futuro, a medida que el marco crece en popularidad como un Angular viable.
9 SPRING JAVA Los expertos en Java pueden utilizar Spring para desarrollar aplicaciones de nivel élite que están diseñadas para manejar bases de usuarios y tráfico masivo. los desarrolladores principiantes, e incluso intermedios, se enfrentarán a una curva de aprendizaje empinada si no tienen experiencia con Java.
10 EMBER PHP aunque es uno de los marcos más potentes y capaces, Ember tiene la capacidad de minimizar la pérdida de tiempo. Además, las soluciones creadas con Ember se pueden convertir en aplicaciones nativas. tiene una curva de aprendizaje considerable cuando comienza por primera vez, un aspecto que se ve reforzado por los módulos que amplían el marco.
